## 使用 Truffle 与 AppChain-Truffle-Migrate 部署合约到 AppChain

- 全局安装 truffle

  - ```shell
    yarn global add truffle
    ```

- 克隆 appchain-truffle-box, 并进入该目录

  - ```shell
    git clone https://github.com/cryptape/appchain-truffle-box.git
    cd appchain-truffle-box/
    rm -rf .git
    ```
